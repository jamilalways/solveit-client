  <h2 style="text-align: center; color: #2c3e50;">
    SolveIt – Problem Solving Marketplace (Clouad ai)
  </h2>

  <p>
    <strong>SolveIt</strong> is a web-based platform where users can post real-life problems 
    and get solutions from skilled individuals in exchange for money. It connects 
    <strong>clients</strong> and <strong>solvers</strong> through a secure and interactive system.
  </p>

  <p>
    Clients can post problems with budget and deadlines, while solvers can submit bids 
    and provide solutions. The system includes chat, payment (escrow), dashboards, 
    and a review system to ensure trust and quality.
  </p>

  <p>
    The frontend is built using a modern, component-based architecture with React, 
    ensuring scalability, reusability, and a clean user interface.
  </p>

  <hr>

  <h3 style="color: #34495e;">📁 Frontend Structure (React)</h3>

  <pre style="background: #f4f4f4; padding: 15px; border-radius: 8px; overflow-x: auto;">
client/
│
├── public/
│   └── index.html
│
├── src/
│   ├── assets/        # images, icons, logos
│   ├── components/    # reusable UI components
│   ├── pages/         # full pages (routes)
│   ├── layouts/       # navbar, sidebar layouts
│   ├── routes/        # routing setup
│   ├── services/      # API calls
│   ├── context/       # global state (auth)
│   ├── hooks/         # custom hooks
│   ├── utils/         # helper functions
│   ├── styles/        # global styles
│   ├── App.jsx
│   └── main.jsx
│
└── package.json
  </pre>

  <p>
    This structure helps maintain a clean and scalable frontend, where components are reusable,
    logic is separated, and the application is easy to manage and extend.
  </p>
